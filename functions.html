Function Writing Guide

📥 Available Variables

  • params - Object containing parameters passed to the function
  • data - Object containing resources. Access as data.resource_name

📤 Return Value

Set result = your_value to return a value.

📚 Example: Filter Data

# Filter users with age > 21
users = data.get('users', [])
adults = [u for u in users if u.get('age', 0) > 21]
result = adults

📚 Example: Use Parameters

# Calculate total with discount
items = data.get('items', [])
total = sum(item.get('price', 0) for item in items)
discount = params.get('discount', 0)
result = total * (1 - discount / 100)

📚 Example: Transform Data

# Add full name field
users = data.get('users', [])
for user in users:
    user['full_name'] = f"{user.get('first_name', '')} {user.get('last_name', '')}"
result = users

📚 Available Python Features

  • List comprehensions: [x for x in items]
  • Dict operations: {k: v for k, v in items}
  • Built-in functions: sum(), len(), max(), min(), sorted()

📦 Import Standard Library

You can import Python standard library modules:

import json
import re
import datetime
import math
import random
import collections

# Use them in your code
today = datetime.date.today()
matches = re.findall(r'\d+', 'abc123def456')
result = math.sqrt(16)

📋 Logs

Use print() statements to log debug info. They will appear in the test results.

# Debug your function with print()
items = data.get('items', [])
print(f"Found {len(items)} items")
for item in items:
    print(f"Processing: {item}")
result = items
